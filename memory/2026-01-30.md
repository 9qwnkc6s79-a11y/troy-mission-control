### SOP: Setting Up a New Secret Menu Item in Toast

**Location:** Toast POS → Menus → New Menu → Secret Cold Lattes
**Restaurant:** Prosper (Restaurant ID: 260183)

#### Step 1: Create the Item
- Navigate to Secret Cold Lattes menu group
- Add new item with name
- Pricing: Inherited from group (Regular 16oz: $5.93, Large 20oz: $6.45)

#### Step 2: Add New Modifiers to Shared Groups (if needed)
- Navigate to the modifier group (e.g., "Recipe Flavors (Iced Latte)")
- Click "Add" at bottom of modifier list
- Type new modifier name in the empty row
- Click "Add" again for additional modifiers
- Save the modifier group
- **⚠️ WARNING:** Adding modifiers to shared groups affects ALL items using that group

#### Step 3: Set Recipe Defaults on the Item
On the item page, scroll to "Selected modifiers will be set by default" section:
1. **Enable Override** — Check the "Override?" checkbox for each modifier group where you need defaults
2. **Check the specific default modifiers** — Once override is enabled, the checkboxes in "Default modifiers" become clickable
3. Save the item

**Technical note:** The Override checkboxes and default modifier checkboxes are in `table[5]` (the 6th table on the page). Override is in column 2 (cells[1]), defaults in column 3 (cells[2]). Checkboxes are disabled until Override is checked.

### JavaScript Method for Setting Defaults (Bulk)
When on an item page, use this approach:
```
// 1. Find defaults table (table index 5 on item page)
const table = document.querySelectorAll('table')[5];

// 2. Enable Override checkboxes for needed rows
// Row indices: 1=Alt Milk, 2=Other Coffee, 3=Shots, 4=Other, 5=Healthy Adds, 6=Recipe SC, 7=Recipe Flavors, 8=Recipe Toppings
const overrideCell = table.querySelectorAll('tr')[ROW].querySelectorAll('td')[1];
overrideCell.querySelector('input[type=checkbox]').click();

// 3. Check specific default modifiers
const defaultCell = table.querySelectorAll('tr')[ROW].querySelectorAll('td')[2];
const inputs = defaultCell.querySelectorAll('input[type=checkbox]');
inputs[INDEX].click(); // INDEX corresponds to modifier position in alphabetical list

// 4. Save via the Save link at top
```
